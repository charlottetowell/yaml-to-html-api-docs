Charlotte's Chocolate Factory API

API for managing chocolate products, orders, and factory operations

api.charlotteschocolatefactory.com/v1

CHOCOLATES

GET /chocolates Returns a paginated list of chocolate products
OPTIONS /chocolates Returns the allowed HTTP methods and other CORS headers
POST /chocolates Add a new chocolate product to the catalog
GET /chocolates/{chocolateId} Returns details of a specific chocolate product
PUT /chocolates/{chocolateId} Update chocolate product
PATCH /chocolates/{chocolateId} Partially update chocolate product
DELETE /chocolates/{chocolateId} Delete chocolate product

ORDERS

POST /orders Create a new order
GET /orders/{orderId} Get order details
GET /users/{userId}/orders Get user's order history

USERS

GET /users/{userId}/orders Get user's order history
