Charlotte's Chocolate Factory API

API for managing chocolate products, Orders, and factory operations

Base URL https://api.charlotteschocolatefactory.com/v1

Chocolates

Operations about chocolate products

GET /chocolates List all chocolate products
Returns a paginated list of chocolate products

Parameters

URL Parameters
Name Location Type Description
page query integer Page number for pagination
limit query integer Number of items per page
category query string Filter by chocolate category

Responses

Response 200

Successful operation

Schema
{
  "Chocolates": [
    {
      "id": (integer),
      "name": (string),
      "category": (string),
      "price": (number),
      "description": (string),
      "inStock": (boolean)
    }
  ],
  "total": (integer)
}

Response 400

Invalid parameters

OPTIONS /chocolates Get chocolate endpoint options
Returns the allowed HTTP methods and other CORS headers

Parameters

No parameters required.

Responses

Response 200

Successful operation

POST /chocolates Create a new chocolate product
Add a new chocolate product to the catalog

Parameters

Request Body Schema
{
  "name": (string) *,
  "category": (string) *,
  "price": (number) *,
  "description": (string),
  "ingredients": [ (string) ]
}

Responses

Response 201

Chocolate product created successfully

Response 400

Invalid input

Response 401

Unauthorized

Apikey Authentication

JWT token in the format: Bearer {token}
